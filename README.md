## Prompt pour créer des QCM (idéal pour des questions avec des données chiffrées)
#
⚙️ Caractéristiques techniques principales
#
► Ce prompt fonctionne très bien avec les IA **Deepseek** et **Claude.ai** (un **compte Google** permet d'y accéder **gratuitement**)
#
► **AUCUNE COMPÉTENCE EN PROGRAMMATION N'EST REQUISE**
#
► A la fin, il suffit de télécharger le fichier créé : **un seul fichier HTML/CSS/JS utilisable sur n'importe quel navigateur**.
#
► Une fois le fichier téléchargé, **aucune connexion Internet n'est requise**.
#
► Idéal pour créer des QCM en mathématiques, gestion, finance, calculs commerciaux, statistiques, etc ...
#
► Créé pour des enseignants, des formateurs, des étudiants, etc...
#
**1. Structure et fonctionnement**
#
✅ Format QCM : 4 réponses possibles par question, une seule correcte
#
✅ Single-file : Tous les codes (HTML, CSS, JS) sont regroupés dans un seul fichier (pour une facilité d'utilisation)
#
✅ Génération aléatoire : Des **données variables pour chaque nouvel exercice (car JS intégré)**. On peut refaire l'exercice avec des **données différentes à chaque fois**, ce qui permet de consolider progressivement les acquis. **Cette répétition variée favorise l'apprentissage et aide à maîtriser pleinement les concepts abordés**
#
✅ Bouton de régénération : **"Refaire un autre exercice du même type"** (orange)
#
**2. Système de validation**
#
✅ Validation globale : Bouton **"Valider mes réponses"**. Pour les mauvaises réponses l'IA donne les bonnes réponses et explique la démarche qu'il aurait fallu suivre
#
**3. Informations complémentaires**
#
✅ Modifier la **ligne 19** du prompt pour **l'adapter à vos besoins** (Donner le thème du QCM)
#
✅ L'idéal serait de **fournir 2 pièces jointes** lors de l'utilisation de ce prompt : un **énoncé** et un **corrigé**. L'IA comprendra tout de suite **"la règle du jeu"** et les QCM fournis seront de meilleure qualité et plus pertinents. Autre alternative : une **capture d'écran de l'énoncé et du corrigé**.
#
✅ Pour les utilisateurs maîtrisant les bases du HTML et du CSS, possibilité de personnaliser ou de modifier le fichier dans le bloc notes (Pensez à enregistrer le fichier **en rajoutant l'extension .html** [en dessous sélectionner **"Tous les fichiers"** avant d'enregistrer])
#
✅ Pour modifier le fichier, au lieu d'utiliser le bloc notes, il est possible d'utiliser le logiciel **Notepad++** ► 
https://notepad-plus-plus.org/
