⚙️ Caractéristiques techniques principales

► Ce prompt fonctionne très bien avec les IA deepseek et claude.ai (un compte Google permet d'y accéder gratuitement)
► AUCUNE COMPETENCE EN PROGRAMMATION N'EST REQUISE
► A la fin, il suffit de télécharger le fichier créé : un seul fichier HTML/CSS/JV utilisable sur n'importe quel navigateur
► Une fois le fichier téléchargé, aucune commexion Internet n'est requise
► Idéal pour créer des QCM en mathématiques, gestion, finance, calculs commerciaux, etc ...

1. Structure et fonctionnement

✅ Format QCM : 4 réponses possibles par question, une seule correcte
✅ Single-file : Tous les codes (HTML, CSS, JS) dans un seul fichier
✅ Génération aléatoire : Données variables pour chaque nouvel exercice (car JS intégré)
✅ Bouton de régénération : "Refaire un autre exercice du même type" (orange)

2. Système de validation

✅ Validation immédiate : Entrée pour chaque réponse
✅ Validation globale : Bouton "Valider mes réponses"

3. Informations complémentaires
   
✅ Modifier la ligne 19 du prompt
✅ L'idéal serait de fournir 2 pièces jointes lors de l'utilisation de ce prompt : un énoncé et un corrigé. L'IA comprendra tout de suite "la règle du jeu" et les QCM fournis seront de meilleure qualité
✅ Pour les utilisateurs maîtrisant les bases du HTML et du CSS, possibilité de personnaliser ou de modifier le fichier dans le bloc notes (Penser à enregistrer le fichier en rajoutant l'extension .html [en dessous sélectionner "Tous les fichiers" avant d'enregistrer])
✅ Pour modifier le fichier, au lieu d'utiliser le bloc notes, il est préférable d'utiliser le logiciel notepad++
